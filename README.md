# Synapta

Synapta is a new technology AI-driven application that enhances productivity for all users across the Internet. Synapta was used specifically to ensure efficiency by giving preferences and suggestions for users based on memory and their data. With user input, customization, structured screenshots, Synapta is tailored and is learned based on your own data. It searches through tabs, applications, and other events on the computer, with the Gemma 3 model learning and training after each screenshot is taken. With optional human input, the application ensures the user still has complete full control, without the AI deciding everything. Synapta also ensures privacy to make sure data is encrypted, with the chatbot not knowing all the user's personal information. This app will be in your workspace and can be activated/deactivated simply with the click of a button. In summary, Synapta is an application that is used greatly to benefit humanity, allowing those to complete and accomplish meaningful tasks without forgetting or even realizing they still had to finish up an assignment.
